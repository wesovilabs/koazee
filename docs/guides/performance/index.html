Performance improvements in Koazee v0.0.2

Benchmark in Koazee v0.0.2.

Why

After releasing Koazee Titi (v0.0.1) I was asked to publish some benchmark. As you know, Koazee is based in reflection and this is even more expensive than I expected before writing the tests.

Benchmark tests helped me to do big performance improvements in v0.0.2 as we will see in this guide.

Running benchmark tests locally

Benchmark tests are available in Koazee repository 

git clone https://github.com/wesovilabs/koazee.git

Once you checkout the repository, you just need to run the below command 

make benchmark

The above code will run benchmark tests for all the operations, if you just want to run the benchmark for a single operation you can do it as shown below 

make benchmark op=add

Available values for op argument are: add, at, contains, duplicates, filter, first, last, maps, reduce

Benchmark: stream:add

About the tests
